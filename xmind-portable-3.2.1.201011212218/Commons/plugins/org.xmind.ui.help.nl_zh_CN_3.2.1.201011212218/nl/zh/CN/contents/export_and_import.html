导出

XMind支持多种格式的导出，例如FreeMind文件，TXT，HTML以及JPEG和PNG格式的单张或者多张图片等等，以及XMind2008工作簿文件。方法如下：

导出对话框

按照下列步骤将思维图导出成FreeMind文件：

  1. 在菜单栏选择"文件 > 导出"；
  2. 在导出对话框中选择"FreeMind文件"；
  3. 点击"下一步"；
  4. 设置文件保存地址；
  5. 点击"完成"结束。

按照下列步骤将思维图导出成TXT文件：

  1. 在菜单栏选择"文件 > 导出"；
  2. 在导出对话框中选择"纯文本文件"；
  3. 点击"下一步"；
  4. 设置文件保存地址；
  5. 点击"完成"结束。

按照下列导出自定义图标：

  1. 在菜单栏选择"文件 > 导出"；
  2. 在导出对话框中选择"图标包"；
  3. 点击"下一步"；
  4. 选择导出的图标组，并设置文件保存地址；
  5. 点击"完成"结束。

按照下列步骤导出HTML文件：

  1. 在菜单栏选择"文件 > 导出"
  2. 在导出对话框中选择HTML；
  3. 点击"下一步"；
  4. 选择文件保存位置；
  5. 在预览对话框中设定是否需要：缩略图、备注、自由主题、标签、图标、图片、联系以及超链接等等；
  6. 点击"完成"结束。

按照下列步骤导出图片：

  1. 在菜单栏选择"文件 > 导出"；
  2. 点击"下一步"；
  3. 在导出对话框中进行下列设定：
    • 导出的文件名称和位置。
    • 图片的文件格式（您可以选择JPEG或者PNG）。
  4. 点击"完成"结束。

按照下列步骤将思维图导出成XMind2008工作簿文件：

  1. 在菜单栏选择"文件 > 导出"；
  2. 在导出对话框中选择"XMind 2008 工作簿"；
  3. 点击"下一步"；
  4. 设置文件保存地址；
  5. 点击"完成"结束。

 

导入

您可以直接导入FreeMind以及MindManager文件至XMind中打开，然后编辑。步骤很简单，如下：

按照下列步骤导入FreeMind文件：

  1. 在菜单栏选择"文件 > 导入"；
  2. 选择"FreeMind 0.8/0.9"，点击"下一步"；
  3. 选择需要导入的文件；
  4. 点击"完成"结束整个过程。

按照下列步骤导入MindManager文件：

  1. 在菜单栏选择"文件 > 导入"；
  2. 选择"MindManger 5/6/7"，点击"下一步"；
  3. 选择需要导入的文件；
  4. 点击"完成"结束整个过程。

按照下列步骤导入XMind 2008工作簿文件：

  1. 在菜单栏选择"文件 > 导入"；
  2. 选择"XMind 2008工作簿"，点击"下一步"；
  3. 选择需要导入的文件；
  4. 点击"完成"结束整个过程。

按照下列步骤导入XMind图片包文件：

  1. 在菜单栏选择"文件 > 导入"；
  2. 选择"XMind图片包"，点击"下一步"；
  3. 选择需要导入的.XMP文件，或者含有图片文件夹；
  4. 点击"完成"结束整个过程。
