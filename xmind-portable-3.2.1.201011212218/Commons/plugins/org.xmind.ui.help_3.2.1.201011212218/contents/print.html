Printing a map in XMind is a simple feature. You can just following the few steps:

  1. Select a map.
  2. Open the print preview dialog via one way following:
    • Choose 'File > Print' on the menu.
    • Use the shortcut 'Ctrl+P'.
  3. Change the Page settlement: Margin size and unit, Header/Footer text.
  4. Click 'Next' and choose the printer.
  5. Click 'OK'.

Note:

  1. The map is set to be printed horizontally no matter choosing Landscape or Portrait.
  2. XMind does not have the feature to let you print multiple maps together in one paper even they are in one workbook.
