Rich text notes are used to record complementary topics details. You can easily add and format the existed notes. Here's how:

To Add notes to a topic:

  1. Select a topic.
  2. Open the pop-out dialog with one of following ways.
    • Click notes icon on the toolbar.
    • Right-click the mouse, select 'Notes'.
    • Press 'F4'.
  3. Type the context in this pop out notes dialog.
  4. Click "Ctrl+Enter(Command+Enter)" to save the notes and close pop out dialog.

To Format existed notes:

  1. Open the Pop-out notes dialog, or Notes View.
  2. Use notes toolbar or Context Menu to do following changes:
    • Text font, size, type, alignment, color.
    • Background color.
    • Insert Images.
    • Insert Hyperlink
  3. If you type the sentence like "http://www.xmind.net" in the notes, XMind will format it as a hyperlink automatically.

Note: You can just insert local image in Notes.
