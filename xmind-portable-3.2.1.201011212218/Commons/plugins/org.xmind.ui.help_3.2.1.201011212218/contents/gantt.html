Gantt View is a type of bar view that illustrates a project schedule. Gantt view illustrates the start and end dates, priority of select tasks. You can also change some elements at this view easily. Here's how

  1. Open map.
  2. Open Gantt View.
    • Only topics that have Taskinfo will be seen here.
    • Every bar is a task. All tasks are list by Priority.
    • You can dag and move the task slide to adjust start and end dates.
    • While you hover mouse on a task, you could see all its details, such as assignee.

Note:

  1. All changes in the Gantt View will be saved as topic' taskinfo.
  2. In TaskInfo View, there is a hyperlink, linked to Gantt View.
