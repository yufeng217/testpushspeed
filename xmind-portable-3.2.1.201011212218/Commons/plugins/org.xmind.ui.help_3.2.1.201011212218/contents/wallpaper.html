Wallpaper is used to show some special meaning or make map beautiful. You can easily to add wallpaper. Here's how:

To Add the wallpaper:

  1. Select map.
  2. Open Properties View.
  3. Click 'Wallpaper...' and select wallpaper in the pop-out dialog.
  4. OR, click 'Select local image..' and select your own image in the pop-out dialog.

Wallpaper list dialog

To Modify and delete the existed wallpaper:

  1. Select map.
  2. Open Properties View
    • Click the delete icon in the properties view to delete added wallpaper.
    • Change wallpaper opacity by imputing number or moving the slider.
