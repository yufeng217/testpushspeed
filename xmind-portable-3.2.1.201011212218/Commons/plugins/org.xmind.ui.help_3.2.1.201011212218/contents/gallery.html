XMind includes hundreds of high-quality images as the Gallery. You can easily to add images into map, change, format or delete added images. Here's how to:

Add image to topic:

  1. Select the topic(s).
  2. Open the Gallery View; or click image icon on the toolbar and select 'From Gallery'.
  3. Choose and click image under your requirements.

Note: you can select multiple topics and add one image at the same time.

 

Format the added image:

  1. Select the topic.
  2. Select the added image.
  3. Right-click mouse and select properties to change size of this image. Or directly drag the to modify.

Change image:

  1. Select the topic.
  2. Open Gallery View; or click image icon on the toolbar and select 'From Gallery'.
  3. Choose another image and click it.

Delete image:

  1. Select the topic.
  2. Select the image.
  3. Click 'delete' key on the board; or Right-click mouse and choose 'delete'.
