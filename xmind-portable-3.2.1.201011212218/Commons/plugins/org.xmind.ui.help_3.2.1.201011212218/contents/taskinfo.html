In XMind Pro, we can add more detailed information to a topic in Taskinfo view. This can help us to have the good performance in Task Management. We can add People who is responsible for this task, start date, end date, task's priority, and the task Progress. Here's how:

  1. Select a topic.
  2. Open TaskInfo View.
  3. You can see and change following information in the view:
    • Task: this is the content of selected Task;
    • Assigned to: Select or input the people who are responsible for this task;
    • StartDate and EndDate: select the proper date for this task. You can even set the exactly time;
    • Progress: move the slide to change task progress;
    • Priority: select proper priority level for this task in the drop-down list;
    • Check Point: enable this check box will now show Progress information in the Gantt Chart View.

Note: All topic's taskinfo will be shown in Gantt View too.
