Collaborative Minds

XMind enables you to attach external files in any formats to a single topic and the attachments will appear as its subtopics. The content of these subtopics are the same as the file name of attached files. In this way you can enrich your ideas by connecting a variety of vivid pieces of information together.

Two different manners to add the attachment:

The first one:

  1. Select the topic.
  2. Open the Attachment dialog via any way of following:
    • Choose "Insert > Attachment" in the menu.
    • Click 'attachment' icon on the toolbar.
  3. Navigate to the file(s), and click 'OK'.
  4. Added file(s) will be the subtopic(s) of target topic.

 

The second one:

  1. Select the file(s) you want to add into XMind.
  2. Drag the file(s) into the map.
  3. Drop file(s) to the target topic as its subtopic.
  4. Drop file(s) to the blank place as map's floating topic(s).

Note: Because all attachments are presented as topics, so you can move, modify, and delete them easily.

 

Open, and save existed attachment:

  1. Select the attachment topic.
  2. Right-click to open the context menu.
  3. You can:
    • Click the 'Open Attachment' to open this attachment.
    • Select 'Save Attachment as...' to save this attachment to other place.
