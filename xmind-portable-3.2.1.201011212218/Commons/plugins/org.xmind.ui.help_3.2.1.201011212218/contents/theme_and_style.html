Theme

XMind has designed six professional themes for users to choose from, which can be applied to any map in just one easy step. If these six themes are not enough, you can design your own special theme and keep it in your XMind. Here's how:

To Change Theme

  1. Open Theme View.
  2. Double-click the theme you like.
  3. Current map's theme will be changed.

To Extract Theme

  1. Design your own theme by choosing color, shape, line, background.
  2. Choose " Tools > Extract Theme..." on the menu.
  3. Name this new theme in the "Theme View".
  4. Now you can apply this theme to other maps.

Note:

  • A theme is a collection of map colors, shapes, line patterns, topic sytle, and boundary style.
  • Eveny theme can only extract one kind topic style, boundary style.

 

Style

Style is a group attributes which can be saved, repeated applied, and delete to a object. You can apply, create four kinds styles, Topic, Boundary, Relationship, and Map, in Style view easily. Here's how:

To Apply style to object(s):

  1. Select the object(s).
  2. Open Style View.
  3. Choose proper style, and double click.
  4. Selected object(s) will be applied that style.

Note:

  • This object can be topic, boundary, relationship, or map.
  • You can just choose one kind one time.
  • You can select multiple objects one time.

To Create personal style:

  1. Select a topic, boundary, relationship, or map.
  2. Format it with the attributes you like.
    • Topic: topic text, shape, and color.
    • Boundary: boundary shape, color, transparency, line style, color, width.
    • Relationship: relationship shape, line style, color, and text.
    • Map: background color, multi-colored branches, and tapered lines.
  3. There are two manners to extract style:
    1. Open Style View > click "Extract Icon" on the view toolbar
    2. Choose "Tools > Extract Style" on the menu.
  4. Name the extracted view.

Notes: You can only extract one kind style every time.

To Edit existed style:

  1. Select the style.
  2. You can do following changes:
    • Click "F2" to rename it.
    • Click "Delete" to delete it.
