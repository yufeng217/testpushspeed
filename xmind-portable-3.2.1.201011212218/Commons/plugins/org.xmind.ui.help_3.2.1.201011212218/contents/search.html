You can search text and images online in XMind. Here's how:

To search text online:

  1. Select one or more topic(s).
  2. Choose "Tools > Search" on the menu.
  3. You will see the search result in Web Browser view.
  4. Select any texts and/or URL, and drag into map.


To search image online:

  1. Select topic.
  2. Click image icon on the toolbar, and choose "From Web" in the drop-down list menu.
  4. Select proper image(s) from Web Browser view, and drag and drop into map.
  5. Image can be dropped into existed topic, or into blank place of map as floating topic.

Note:

  • Topic Content will be keywords for searching text and image.
  • You can change the search engine easily under following steps:
    1. Choose "Edit > Preferences.."
    2. Select "Search Engines"
    3. Choose the search engines under your requirements.
