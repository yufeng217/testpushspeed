Label is a small text tag that is attached to topics, and typically used for simple annotation and categorization. A topic can have multiple labels, separated by commas. Added labels will be list in the marker view. You can filter the topics with label.

You can easily create, edit, and delete existed label easily. Here's how:

To add a label:

  1. Select a topic.
  2. You have three manners to open label field:
    • Click the label icon on toolbar.
    • Right-click the mouse to open the context menu and click 'label'.
    • Use shortcut key 'F3'.
  3. Type the label text in the field below selected topic.
  4. Click 'Enter'.

Every topic can have multiple labels. Just use comma to separate them. Ant the labels will be in alphabet order.

To edit a label:

  1. Select topic.
  2. Open the label field.
  3. Edit the existed label.
  4. Click 'Enter'.

To delete a label:

  1. Select topic.
  2. Open the label field.
  3. Delete this existed label.
  4. Click 'Enter'.

Note: You can not create, edit, or delete a label for multiple topics at the same time.
